# 一醒
方寸一句，豁然一醒

一款优雅的桌面名言显示插件，每日为您呈现智慧与灵感。精选海量名言警句，涵盖文学、影视、哲学、诗词等12个分类，让您的桌面充满智慧光芒。

## 核心功能：
* 📚 12个精选分类，涵盖古今中外
* ⚙️ 自定义显示效果与切换间隔
* 🌙 优雅的毛玻璃背景效果
* 💾 智能缓存，离线可用
* 📱 响应式设计，适配各种屏幕
